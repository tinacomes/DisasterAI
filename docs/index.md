# Potential deprivation across European cities — results site

Placeholder for the public data-story companion (static export of the
per-city maps and cross-city figures produced by `src/depacc/viz/`).

Planned pages, populated as the pipeline runs per city:

- **Compounding deprivation maps** — bivariate everyday × emergency typology
  per city (HH = deprived on both).
- **Everyday vs emergency divergence** — every city as a point in the
  Gini-vs-Gini plane.
- **The size gradient** — divergence and clusters ordered along the FUA
  population gradient (cross-sectional, space-for-time reading).

Enable GitHub Pages from `docs/` once the project is split into its own
repository (see README → "Note on hosting").
